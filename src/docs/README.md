# InsightX Pro — Screenshots / Documentation

This folder is reserved for dashboard screenshots and any additional documentation assets.

## How to Add Screenshots to README

1. Take a screenshot of your running dashboard
2. Save it in this folder as `dashboard-overview.png`, `dark-mode.png`, etc.
3. Reference it in `README.md`:

```markdown
![Dashboard Overview](./src/docs/dashboard-overview.png)
```

## Recommended Screenshots
- `upload-screen.png` — The landing/drag-drop screen
- `dashboard-dark.png` — Dashboard in dark mode with charts
- `dashboard-light.png` — Dashboard in light mode
- `data-cleaning.png` — Data cleaning page with heatmap
- `reports.png` — Reports page
